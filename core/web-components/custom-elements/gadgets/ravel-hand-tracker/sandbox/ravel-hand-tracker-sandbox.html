attributes & events

landmark — MediaPipe landmark index to track (0–20). Default 8 (index fingertip).
facing — user (front cam, auto-mirrored) or environment (rear cam). Default user.
autoplay — boolean, starts tracking immediately on connect.
arm-id / arm-id-right — ID of a ravel-control-arm to drive with left/right hand data.
signal-out — message channel name; broadcasts { x, y, landmarks } for every frame.

Events (bubbling, composed): pinch-start and pinch-end — detail: { slot: 0|1 } (slot 0 = leftmost hand).
