front camera — useful on desktop with a screen displaying a QR code

attributes & events

facing — environment (rear/default) or user (front).
fps — scan rate in frames per second. Default 10.
qrbox — scan region: auto (full frame), min (60% of smaller dimension), or a pixel number.
autoplay — boolean, starts on connect.

Requires Html5Qrcode to be loaded as a global before this element's module script.

Event (bubbling, composed): qr-detected — detail: { text: string }. Fires once per unique value; resets on each new start.
