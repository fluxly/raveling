ravel-gadget

BLE gadget — connects via Web Bluetooth
Serial gadget — connects via Web Serial (USB)
broadcast events
ravel-gadget channel
notes
Clicking Connect opens the browser's native device picker.
BLE: looks for devices with name prefix "Ravel", service 0xff00.
Serial: filters for nRF52 (0x2886/0x8045) and ESP32 (0x303A/0x1001).
Requires HTTPS or localhost; Web Bluetooth/Serial must be enabled in the browser.
