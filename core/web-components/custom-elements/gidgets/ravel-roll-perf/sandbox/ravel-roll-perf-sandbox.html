attributes & interaction

length — duration in units.
unit-width — px per unit (controls display width).
quantum — snap step used when resizing.
color — block background color.
label — text shown in the center.
selected — highlights the block (full opacity).
docked — height fills 100% of parent track lane.
no-click — disables all pointer interaction.

Interaction zones:
Left / right handles → ew-resize to scale length.
Center → drag to move (broadcasts undock after threshold) · click to toggle selected.
Cmd+click → remove.
