attributes

unit-width — px per roll unit (1 unit = 1/128th note).
quantum — snap grid in units.
h — track height in px.
label — text shown inside the lane (right-aligned).
color — color applied to all docked perfs.
tick / tick-mul — minor tick interval and major emphasis multiple.

Interaction: shift+click a lane to paint a note · drag a note to move it.
API: dockPerf(el, unitPos) — dock a perf programmatically.
Events: ravel-roll-track (cmd: state-change) · ravel-roll-perf (cmd: undock).
