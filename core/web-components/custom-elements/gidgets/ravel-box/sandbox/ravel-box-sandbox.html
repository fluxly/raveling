ravel-box

Up to three concentric border rings + colored center. Each ring is pixel-size px thick.

1, 2, and 3 borders — pixel-size="10"
fluoro palette — three borders each
pixel-size variation — 4 / 8 / 12 / 16 px
slotted content inside the center panel
HELLO ravel-box
absolute positioning via x / y
attributes

w / h — total box size in px. Defaults: 100 × 100.
x / y — position offset when the parent is position: relative.
border-1 — outermost ring color. Default #000000. Empty = no ring.
border-2 — middle ring color. Default empty.
border-3 — innermost ring color. Default empty.
pixel-size — thickness of each active ring in px. Default 10.
color — fill color of the center panel. Default #ff0000.
