ravel-fluxum

Glowing circular avatar — emoji, SVG, or glow-only. Driven by attributes or transform().

color + icon — fluoro palette
sizes — 25 / 50 / 75 / 100 / 120 px
show-hint — speech bubble
use-svg — embedded SVG with color tint
transform(x, y, angleRad) — animated orbit
angle attribute — static rotation (degrees)
attributes & API

icon — emoji or text inside the circle (default 🐹).
size — diameter in px (default 50).
x / y — centre position in px relative to nearest positioned ancestor.
angle — rotation in degrees for static placement.
color — hex color for fill + glow effect.
show-hint — show a speech bubble above the avatar.
use-svg — replace emoji with the embedded Fluxum SVG mark.

Method: transform(x, y, angleRad) — update position + rotation each physics frame. Angle is in radians (physics engine convention).
