gap — any valid CSS gap value. Defaults to 5px. Always 3 columns; rows grow automatically.
scale — CSS zoom factor. Shrinks both the rendered size and the layout footprint, so a scaled nonagram fits exactly inside a parent cell. The center cell above has its scale calculated automatically: scale = cell_width / outer_width.
