attributes & API

snap — 1 (default) enables scroll-snap; 0 disables it.
arrows — 1 (default) shows prev/next buttons; 0 hides them.
page-size — pixels to scroll per arrow click. viewport (default) uses the visible width.

CSS custom properties: --gap, --pad, --btn-size, --btn-bg, --btn-fg, --btn-radius, --fade-width.

Methods: scrollByPage(1 | -1), scrollToIndex(n).
