ravel-filmstrip

Frame-by-frame image player — slot <img> elements as frames

default — 10 frames, fps=12, loop
play-sound event log
waiting for sounds…
no loop — stops at end
attributes & API

fps — frames per second (default 6).
loop — boolean; loops playback at end.

Methods: play(), pause(), stop(), next(), previous().
addSoundToCurrentFrame(token) — attach a string token (e.g. emoji) to the current frame; fired as play-sound events during playback.
clearSoundInCurrentFrame() — remove tokens from the current frame.

Event: play-sound (CustomEvent, bubbles) — detail: { token: string, frameIndex: number }.
