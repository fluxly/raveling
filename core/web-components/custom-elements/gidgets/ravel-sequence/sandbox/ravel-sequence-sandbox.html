FRAME 01 / 05

ravel-sequence

A full-viewport navigable stack. Each slotted child is a frame — only one is visible at a time. Use the buttons below or the controls panel to navigate.

FRAME 02 / 05

Breadcrumbs

The pill shows dots or numbers via breadcrumb-style. Dot color is set with breadcrumb-color. Try switching in the controls.

FRAME 03 / 05

Navigation

Prev / Next labels are set via prev-label and next-label. The buttons disable automatically at the first and last frame.

FRAME 04 / 05

Jump

Set the index attribute at any time to jump to a frame programmatically. Changes broadcast on the ravel-sequence channel with cmd change.

FRAME 05 / 05

Pill Position

The nav pill position is controlled by pill-bottom and pill-width. The icon attribute sets the glyph shown in the center of the pill.
