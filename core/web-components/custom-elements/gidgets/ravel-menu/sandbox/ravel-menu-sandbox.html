type="dropdown" (default) — click trigger opens floating panel
type="context" — right-click inside the zone below
right-click anywhere in here
type="bar" — items displayed horizontally, no trigger or panel

attributes & events

type — dropdown (default) · context · bar
label — trigger button text for dropdown (default: "Menu")
open — boolean attribute to force-open the panel

Slot any <button data-cmd="...">, <a>, or <hr> as items.
disabled on a button dims it and prevents activation.

Event: menu-select → detail.cmd (from data-cmd), detail.item (the element).
Also broadcasts on channel ravel-menu with { cmd: 'select', content: { cmd } }.

Keyboard (dropdown / context): ↓/↑ navigate · Escape closes · ↓ on trigger opens.
