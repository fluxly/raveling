setup & attributes

Requires Box2D loaded before the module:
  <script src="/core/libs/Box2d.min.js"></script>

damping — mouse-joint damping ratio (0 = full spring, 1 = heavy drag).
width / height — override world size (auto-detected from layout).

Any HTMLElement can be slotted as a physics body.
x / y attributes set the initial center position in px.
<ravel-fluxum> → circular body, moved via .transform().
All other elements → rectangular body sized from rendered dimensions;
  the world sets position: absolute on them automatically.

Angular velocity is broadcast each frame on the element's id channel
(cmd: 'playback-rate') — wire to <ravel-fluxly-sound-engine>.
