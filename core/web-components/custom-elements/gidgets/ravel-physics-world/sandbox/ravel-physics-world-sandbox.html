damping — mouse-joint damping ratio (0 = full spring, 1 = heavy drag).
width / height — override world size (auto-detected from layout).

Slot <ravel-fluxum> elements with x, y, size attributes.
Physics bodies are registered at setup time; resizing the window updates the offset.

Each frame, angular velocity is broadcast on fluxum-N channels
(cmd: 'playback-rate') — wire to <ravel-fluxly-sound-engine>.
