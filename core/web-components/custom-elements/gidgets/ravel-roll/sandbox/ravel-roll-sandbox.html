attributes

unit-width — px per roll unit (1 unit = 1/128th note).
quantum — snap grid in units.
track-h — height of each pitch row in px.
tick / tick-mul — ruler line interval and emphasis multiple.
length — total roll width in units.
h — component height in px.
tracks — number of chromatic rows (default 87).
show-keyboard — show/hide the sticky piano keyboard strip.
onion-src — URL of an SVG to overlay as a reference layer.

Interaction: shift+click a row to paint a note · drag a note to move it.
Control bar: offset slider · width slider · length slider · MIDI IN · MIDI out.
