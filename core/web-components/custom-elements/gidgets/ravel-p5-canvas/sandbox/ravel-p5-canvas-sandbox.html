ravel-p5-canvas

Runs a p5.js sketch fetched from a URL — instance mode, sandboxed by default

src="./testSketch.js" — file on disk
live editor — edit sketch code and apply
autostart="0" — manual start
ravelp5:loaded / ravelp5:error events
waiting for events…
attributes & API

src — URL of the p5 sketch file to fetch.
w / h — canvas dimensions in px (default 300 × 150).
autostart — 1 (default) runs on connect; 0 waits for reload().
sandbox — 1 (default) shadows window/document inside the sketch; 0 allows global access.

Sketches use instance mode — assign onto p:
  p.setup = () => { p.createCanvas(p.RAVEL_W, p.RAVEL_H); }
  p.draw = () => { p.background(0); }

p.RAVEL_W / p.RAVEL_H — live getters for the component's w/h attributes.

Methods: reload(), stop().
Events: ravelp5:loaded, ravelp5:error (both bubble + composed).
