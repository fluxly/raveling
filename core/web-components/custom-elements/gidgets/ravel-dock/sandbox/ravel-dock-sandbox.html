usage

Any component can register itself by calling this.dock() (from RavelElement).
That sends cmd='dock-me' with the element as content on channel 'ravel-dock'.
The dock reads the element's icon and label attributes.
Use a custom label attribute on <ravel-dock> to support multiple docks per page.
