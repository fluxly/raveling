API reference

label — pub/sub channel the broker subscribes to (default: ravel-midi-broker).
channel — window event channel for parsed MIDI broadcasts (default: midi). Shown below the LEDs.
device — optional device-name filter; empty = accept all devices.

Inbound (ravel-midi-gadget → broker via RavelMessenger.sendMessage):
  cmd: 'raw', content: { data: number[], deviceName?: string }

Outbound (broker → components via window on channel):
  note-on / note-off · { channel, note, velocity }
  cc · { channel, controller, value }
  pitch-bend · { channel, value } (−8192 to 8191)
  program-change · { channel, program }
  aftertouch / poly-aftertouch · { channel, pressure }
  clock · { type: 'tick'|'start'|'stop'|'continue' }
  sysex · { data: number[] }

MIDI out (components → broker → gadget):
  Send cmd: 'send', content: { type, channel?, note?, velocity?, ... } to the broker's label channel.
  The broker encodes raw bytes and calls broadcastMessage('ravel-midi-gadget', 'send', { data }).
